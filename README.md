# INSY7314_Swift_System

## How to Run
1. Clone or Download the repository
2. Open the terminal in VSCode
3. Use the split icon in the terminal to have two terminals open at once
4. Using the left terminal, use the command 'cd' into the 'backend' then run 'npm install' and then 'npm run start'
5. Using the right terminal, use the command 'cd' into the 'frontend' then run 'npm install' and then 'npm run start'
6. This should open the browser with localhost with the frontend displayed otherwise use localhost:3000
